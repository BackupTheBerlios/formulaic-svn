Package formulaic

A web form generation package designed to interoperate with FormEncode.

Formulaic helps you automatically generate the html for web forms. Out of the box, it can handle things like error messages, default values, intelligent detection and rendering of "required" fields, and, above all else, customizable markup. Formulaic acts more-or-less like a wrapper for the FormEncode form validation package, and preserves virtually all of its functionality.

Author: Greg Steffensen

Contact: greg.steffensen@gmail.com
