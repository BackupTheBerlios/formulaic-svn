Module Hierarchy

  • formulaic: A web form generation package designed to interoperate with FormEncode.
    • forms: Simple form class that can be used and customized directly, or subclassed.
    • htmlwidgets: A basic set of Formulaic widgets wrapping the standard html form elements
      • widgetclasses
    • odict: A dict that keeps keys in insertion order...

Class Hierarchy

  • __builtin__.object: The most base type
    • __builtin__.dict: dict() -> new empty dictionary.
      • formulaic.odict.OrderedDict: A class of dictionary that keeps the insertion order of keys.
        • formulaic.forms.BaseForm: A basic formencode-enabled html form, designed to be easily customizable through subclassing.
          • formulaic.forms.RequirementsForm: A form that autodetects whether fields are required, and renders their labels differently if so
          • formulaic.forms.TableForm: A form that is rendered in a simple 3-column html table (label, widget, error)
    • __builtin__.type: type(object) -> the object's type type(name, bases, dict) -> a new type
  • exceptions.Exception: Common base class for all exceptions.
    • formulaic.odict.BadOrderingError: The user tried to set an order that was inconsistant with the current contents of the dict...
  • formulaic.htmlwidgets.widgetclasses.Widget: Abstract base class for widgets to inheirit from...
    • formulaic.htmlwidgets.widgetclasses.Custom: A callable that returns a custom html string, intended for the creation of simple custom widgets
    • formulaic.htmlwidgets.widgetclasses.Input: A callable that can be used to render html input elements of any type
      • formulaic.htmlwidgets.widgetclasses.CheckboxInput: A callable that renders html checkbox input elements
      • formulaic.htmlwidgets.widgetclasses.RadioInput: A callable that renders html radio input elements...
      • formulaic.htmlwidgets.widgetclasses.Select: A callable that renders an html select element, including its options.
      • formulaic.htmlwidgets.widgetclasses.Textarea: A callable that renders html textarea elements
