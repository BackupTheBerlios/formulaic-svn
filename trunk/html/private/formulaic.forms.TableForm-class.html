formTpl

The template for the entire form. Accepts three template parameters: $fields (the final, joined rendering of all the fields of the form), $formAttributes (a single, joined string rendering of all the html attributes to be applied to the form element, such as "action"), and $footer, the rendering of the form footer, which is often just a submit button.
Type:
str
Value:
'''<form $formAttributes>
<table $tableAttributes>

$fields

</table>
</form>'''

normalFieldTpl

the template for rendering fields in "normal" mode (as opposed to "bare" mode). Takes three parameters: $label, $widget and $error.
Type:
